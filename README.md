1. 用户模块
 用户注册、登录、退出
个人信息修改
我的预约、我的订单、取消预约

2. 汉服展示模块
汉服列表、分类展示（唐制、宋制、魏晋、明制、洛阳特色等）。
汉服详情（图片、尺码、租赁价格、可预约时段、库存）。
 搜索（可直接搜索朝代关键字来选）、筛选、排序（价格、热度）。

3. 预约核心模块
选择预约日期、时间段。
选择汉服款式、数量。
选择附加服务（妆造、发型、摄影、道具）。
 提交预约单、填写使用人信息（姓名➕手机号就可）

4. 订单与支付模块
订单生成、订单详情查看。
订单状态跟踪（已预约、已使用、已取消）。
支付模拟（余额支付/线下支付）。

5. 门店与公告模块

洛阳门店信息（地址、营业时间、联系电话）。
预约须知（服装归还时间为当天21:00前，服装如有破损，道具丢失则根据实际情况扣除相应金额）。
留言反馈、在线咨询

三、后台管理功能

1. 管理员模块
管理员登录、权限管理

2. 用户管理
用户列表查询。
用户信息查看。

3. 汉服管理
汉服信息增删改查。
汉服分类管理（朝代，颜色）。
图片上传、库存设置、上下架。

4. 预约订单管理
订单页面查询、详情查看。
订单审核、确认、取消。
标记订单完成。

5. 服务套餐管理
 妆造套餐、摄影套餐、道具租赁增删改查。
6. 数据统计与报表
每日/每月预约量统计。

五、添加功能
1. 洛阳本地化：添加洛阳景点推荐、景点搭配建议
3. 图片批量上传、轮播展示（在汉服展示页面体现）
4. 预约成功短信/邮箱通知（模拟即可）
